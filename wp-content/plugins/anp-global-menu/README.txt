=== Activist Network Global Menu ===
Contributors: Pea, Glocal
Tags: navigation, multisite, global menu
License: GPL3

A Wordpress plugin that adds a global menu to all sub-sites in a multi-site network.

Usage:

* The plugin should be activated network-wide
* The menu to display on network sub-sites can be selected in Network > Settings > Global Menu

Notes:
* At least one menu must exist on the main site in order for this to work
* The global menu is not added to the main site.
